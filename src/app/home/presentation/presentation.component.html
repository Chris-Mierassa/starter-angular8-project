Présentation

Ce template est un template quit de démarrage rapide pour un projet angular 8. il se compose des éléments suivants:

  • SCSS
  • Bootstrap 4
  • NgBootstrap
  • Ngx-toastr
  • Ngx-toastr
  • Ngx-LoadingModule
  • ReactiveFormModule

Le template a été connecté à l'API Firebase de google (à paramétré) qui permet de gérer la connexion. Une fois connecté, vous pourrez accéder à votre Admin et via des requêtes http ajouté ou supprimer des articles. Vous bénéficiez aussi de l'auto loading.
